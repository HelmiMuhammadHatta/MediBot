# 🏥 MediBot — Asisten Kesehatan Pribadi AI

MediBot adalah chatbot kesehatan berbasis **Gemini 2.5 Flash** yang dirancang untuk memberikan informasi medis, nutrisi, dan gaya hidup sehat secara cepat dan menenangkan.

## ✨ Fitur

- 🤖 **Powered by Gemini AI** — Model AI terbaru untuk jawaban medis yang akurat
- 🏥 **Health Topics** — Gejala, Nutrisi, Kesehatan Mental, Obat, Ibu & Anak, Lansia
- 💬 **Conversation History** — MediBot mengingat konteks diskusi kesehatan Anda
- 🎯 **Medical Sidebar** — Navigasi mudah berdasarkan kategori kesehatan
- 🌙 **Modern Medical UI** — Desain "Medical Blue" yang elegan dan profesional
- ⚠️ **Health Disclaimer** — Peringatan otomatis untuk konsultasi dengan dokter

## 🛠️ Tech Stack

| Layer | Teknologi |
|-------|-----------|
| Frontend | HTML5, CSS3, Vanilla JS |
| Backend | Node.js + Express |
| AI Model | Google Gemini 2.5 Flash |
| SDK | @google/generative-ai |

## 🚀 Cara Menjalankan

1.  **Isi API Key**: Masukkan key di file [`.env`](file:///c:/ProjectHacktiv8/.env)
2.  **Jalankan**: `npm start`
3.  **Akses**: [http://localhost:3000](http://localhost:3000)

## 🎭 Persona MediBot

MediBot dikonfigurasi sebagai asisten yang:
- **Empatik & Menenangkan**: Menggunakan bahasa yang ramah dan tidak menakut-nakuti.
- **Informatif**: Memberikan penjelasan terstruktur mengenai kondisi kesehatan.
- **Etis**: Selalu menyarankan konsultasi dengan tenaga medis profesional untuk diagnosis serius.
